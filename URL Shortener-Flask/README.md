# 🔗 URL Shortener (Flask App)

A simple web application that allows users to shorten URLs using multiple services.

---

## 🚀 Features

* 🔗 Shorten URLs using multiple providers
* ✅ URL validation
* ⚠️ Error handling with user feedback
* 📋 Copy-to-clipboard functionality
* 🎨 Clean UI with Bootstrap

## 🛠️ Tech Stack

* Python
* Flask
* pyshorteners
* Bootstrap

---

## 🌐 Usage

1. Enter a URL
2. Select a shortening service
3. Click "Shorten URL"
4. Copy and share the result
